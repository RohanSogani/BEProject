What is xCalibur?


     xCalibur analyses android application both statically and dynamically. In this process, it analyses data flow in the application and conducts penetration testing on the basis of predetermined test cases. It also combines the statistics and create a holistic threat analysis report. Using these reports, we will create a repository of analysed application and rate applications accordingly. Our ultimate aim is to create a standard definition for Android application security.
