HackSense

HackeSense is the Excalibur sword of our project. It aims to dissect, coerce and beat the truth of your application ;). HackeSense modules run Dynamic and Static analysis tests on your application solely designed to expose flaws and faults in application which may lead to data exposure from your smartphone.

Step 1: The application is first subjected to a static analyzer which a list of Android system permissions the application requires to function. Static analysis involves decompilation of apk files to source code and then extraction of permissions from the application manifest file.

Step 2: The application is now subjected a thorough dynamic pen-testing to identify threats to your android phone. These threats include as of now SQL injection but the future scope plans the complete net of the OWAPS Top 10 threat index. These tests are used to determine the extent of secure development practices adopted by the application developers and to expose loopholes.
