IntelliSight

If HackeSense is Batman, IntelliSight is Alfred. The sole purpose of IntelliSight is to generate a human readable report which a user and developer alike can read.

IntelliSight combines outputs of HackeSense and generates a report on the performance of Apps. On completing static and dynamic analysis, the result is converted to a user readable document and is added to the AppWindow as a version of the application. The AppWindow database is kept in squeaky clean and efficient condition by IntelliSight.
